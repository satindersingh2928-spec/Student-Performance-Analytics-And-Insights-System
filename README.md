# 🎓 Student Performance Analytics & Insights System

🚀 A data-driven web application designed to analyze, manage, and visualize student performance using modern analytics techniques.

🔗 **Live Demo:** https://student-performance-analytics-and-pkv1.onrender.com/

---

## 📊 Project Objective

The goal of this project is to build a **data-centric system** that not only manages student records but also provides **actionable insights** from academic data.

This project simulates a real-world scenario where educational institutions can:
- Track student performance
- Identify trends and patterns
- Make data-driven decisions

---

## 🧠 Key Highlights (For Data Roles)

- 📊 Data collection & storage using MySQL
- 📈 Performance analysis (marks, attendance, grades)
- 📉 Trend identification using dashboards
- 📤 Data export for further analysis (Excel/PDF)
- 📊 Integration with Power BI for visualization
- 🔍 Filtering, searching, and pagination (data handling)

---

## 📌 Features

### 👨‍🎓 Student Data Management
- Add, update, delete student records
- Structured data storage
- Search & filter functionality

### 📊 Performance Analytics
- Automatic calculation of:
  - Total marks
  - Percentage
  - Grade classification
- Helps in identifying:
  - High performers
  - Low performers

### 📥 Data Ingestion (Excel)
- Bulk upload student data
- Real-world ETL-like process

### 📤 Data Export
- Export structured datasets to:
  - Excel (for analysis)
  - PDF (for reporting)

### 📈 Visualization
- Power BI dashboards for:
  - Performance distribution
  - Trends & comparisons

### 🔐 Authentication System
- Admin & Student roles
- Secure login with hashed passwords

---

## 🔄 Data Workflow

1. Data is collected (manual entry / Excel import)
2. Stored in MySQL database
3. Processed (calculations for marks, grades)
4. Queried & filtered using Flask backend
5. Exported for reporting (PDF/Excel)
6. Visualized using Power BI dashboards

---

## 📊 Analytical Use Cases

- Identify students below performance threshold
- Analyze attendance vs performance correlation
- Compare performance across courses/years
- Generate reports for academic evaluation
