About The Project

A complete solution for managing, analyzing and visualizing student performance.

The Student Performance Analytics & Insights System is a web-based application designed to efficiently manage student data and analyze academic performance. It provides a centralized platform where administrators can store, update, and monitor student records in a structured and organized manner.

The system follows a role-based authentication mechanism. An Admin can register using a secure Secret Admin Key, and once registered, can log in to access the full system. On the other hand, Students do not register manually — they can log in using their Registration Number as username and a default password provided to them.

For security and flexibility, students also have the option to change their password after logging in. This ensures both ease of access and improved account security.

The system enables real-time tracking of marks, attendance, and overall performance, while automatically calculating total marks, percentage, and grades. It also supports Excel and PDF export for reporting, and integrates with Power BI dashboards to visualize insights and trends effectively.

Overall, this project simplifies student management, reduces manual effort, ensures data accuracy, and provides meaningful insights for better academic decisions.

✅ Automatic total, percentage & grade calculation
✅ Excel & PDF report generation

✅ Password change functionality for students
